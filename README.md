# Thinking Machine Dashboard

AI-powered decision support system that helps users solve complex problems using mental models and structured analysis.

## Features
- Problem management system with CRUD operations
- AI-powered analysis using Google Gemini API
- Mental models visualization
- Structured analysis reports
- Local storage persistence

## Tech Stack
- React 18 + TypeScript
- Tailwind CSS
- Vite
- Google Gemini API

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env.local` file with your Gemini API key:
   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```
4. Run development server: `npm run dev`
